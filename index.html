Loops allow you to repeat a block of code as many times as you need. It’s like telling Python to keep doing something until you say stop.

Using a For Loop

One way to repeat actions is by using a for loop. A for loop is great when you know exactly how many times you want to repeat something. 

greetings = ["Konnichiwa", "Ohayou", "Konbanwa"]
for greeting in greetings:
    print(greeting)
Konnichiwa
Ohayou  
Konbanwa

Here, each greeting in the list is printed one by one. The for loop repeats for each item in the list, printing each greeting.

Functions are blocks of code that perform a specific task and can be used multiple times. They allow us to write code once and reuse it whenever we need to perform that task.

Defining and Calling a Function

Here’s how to define a function that adds two numbers and gives you the result. A function is created using the def keyword, followed by a name and a set of parentheses. Inside of the parentheses you can add parameters to the function, separated with a coma.

If you define a function and you don't call it it, the code inside it will never get executed. So to start their code you have to call them. 

def add_numbers(number1, number2): # (1)
    # Since we called function like this: add_numbers(5, 3) internally its saying
    # number1 = 5, number2 = 3
    result = number1 + number2
    return result # this return goes back to the call portion, saving it in the sum variable

#THE EXECUTION STARTS FROM HERE DOWN c:   
sum = add_numbers(5, 3) # here we are calling the function so we go there (1)
print("The sum of the numbers is: ", sum)

In Python, you can read from and write to files. This is particularly useful in cybersecurity for analyzing logs, detecting anomalies, and extracting useful information from data files.

Read Files in Python

To read a file in Python, you can use the open() function. This function opens a file and allows you to specify the mode you want to use. For reading a file, you would use "r" as the mode. Here’s how it works:

  • Use with open("filename", "r") as file: to open the file. This ensures that the file is properly closed after its suite finishes. In the "as file" u can put whatever name you like, could be as "epicFile" but then on the for loop you'll have to change it too. It's a convention to put "as file" c:
  • You can then iterate through the file line by line using a for loop.
  • Don't really have to understand why its called with open... just know it'll always like that for file management.

Here’s an example of reading a file and printing each line: 

with open("example.txt", "r") as file:
    for line in file:
        print(line.strip())  # strip() removes leading/trailing whitespace

The Strip method is used a lot in file management, it can help you remove stuff from strings.

Writing a Log File

You can create a log file that records events, such as login attempts. This is done using the open() function in write mode ("w"). Here we are creating a file called login_attempts.txt and we adding those lines as text (like the linux << but with python c:). The \n represents a line break in Python c: 

with open("login_attempts.txt", "w") as file:
    file.write("user1: success\n")
    file.write("user2: failure\n")
    file.write("user3: success\n")
    file.write("user4: failure\n")
